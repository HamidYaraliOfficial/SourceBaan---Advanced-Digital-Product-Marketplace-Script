# SourceBaan - Advanced Digital Product Marketplace Script

## English

### Overview

**SourceBaan** is a complete, full-featured PHP-based digital product marketplace script inspired by leading source code selling platforms. It enables you to launch a professional website for selling digital products such as source codes, scripts, templates, plugins, and more — all without requiring a database. Everything is powered by JSON files for simplicity, speed, and easy customization.

This FULL version includes admin panel, user panel, secure downloads, payment integration hooks, advertising system, real-time chat, email OTP verification, advanced captcha, file antivirus scanning, and comprehensive analytics.

### Key Features

- **Professional Design**: Modern, responsive UI/UX with clean product pages and secure download system.
- **Complete Sales System**: Product management (title, description, price, versions, stock), categories, advanced search & filters.
- **User Management**: Registration with email OTP, user dashboard, order history, and role-based access.
- **Shopping Cart & Checkout**: Multi-item cart, dynamic pricing, and payment gateway integration ready.
- **Powerful Admin Panel**: Full control over products, users, orders, site settings, SEO, themes, SMTP, and security logs.
- **Built-in Advertising System**: Create banners, popups, sliders, notifications with scheduling and click/impression stats.
- **Email OTP Verification**: Secure registration using one-time password sent via custom SMTP.
- **Advanced Captcha**: Bot protection without external APIs — works on shared hosting.
- **File Antivirus Scanner**: Automatic scanning of uploaded files for malware, shells, and backdoors with admin reports.
- **Real-Time Support Chat**: Live chat between users and admin with notifications and JSON-stored history.
- **JSON-Based Storage**: No MySQL required — all data stored in editable JSON files.
- **Easy Customization**: Change logo, colors, texts, templates, and domain references directly in code/data.
- **Statistics Dashboard**: Daily, weekly, monthly sales reports with visual charts.

### Requirements

- PHP 7.4 or higher
- Web server (Apache/Nginx)
- Shared hosting, VPS, or local server (XAMPP/WAMP)
- SMTP server for email notifications (optional but recommended)

### Installation

1. Upload all files to your web server.
2. Edit domain references (replace `sourcebaan.ir` and `sourcebaan` with your domain).
3. Configure SMTP settings in the admin panel if needed.
4. Access your site and log in to admin:
   - Email: `admin@sourcebaan.local`
   - Password: `SourceBaan@123`
5. Customize logo, colors, and content via the admin panel.

### Admin Login

- **Email**: admin@sourcebaan.local
- **Password**: SourceBaan@123

### Security Notes

- Change default admin credentials immediately after setup.
- Uploaded files are automatically scanned for threats.
- All downloads are secured and require valid purchase.

## 中文

### 概述

**SourceBaan** 是一款完整的专业数字产品销售平台脚本，基于 PHP 开发，灵感来源于领先的源码销售网站。您可以使用此脚本快速搭建一个专业的数字产品销售网站，包括源码、脚本、模板、插件等 — 无需数据库！所有数据通过 JSON 文件管理，速度快、简单且易于自定义。

此 **FULL 版本** 包含完整管理员面板、用户面板、安全下载、广告系统、实时聊天、邮箱 OTP 验证、高级验证码、文件防病毒扫描和全面统计功能。

### 主要功能

- **专业设计**：现代响应式 UI/UX，精美产品页面和购买后安全下载系统。
- **完整销售系统**：产品管理（标题、描述、价格、版本、库存）、分类、搜索和高级过滤。
- **用户管理**：邮箱 OTP 注册、用户仪表板、订单历史和角色权限。
- **购物车与结账**：支持多商品、动态定价，预留支付网关接口。
- **强大管理员面板**：完全控制产品、用户、订单、站点设置、SEO、主题、SMTP 和安全日志。
- **内置广告系统**：创建横幅、弹窗、滑块、通知，支持定时投放和点击/展示统计。
- **邮箱 OTP 验证**：通过自定义 SMTP 发送一次性验证码，确保安全注册。
- **高级验证码**：无需外部 API 防止机器人 — 在共享主机上正常运行。
- **文件防病毒扫描**：自动扫描上传文件，检测恶意代码、Shell 和后门，并在管理员面板报告。
- **实时客服聊天**：用户与管理员直接对话，支持实时通知，聊天记录保存在 JSON。
- **JSON 存储**：无需 MySQL — 所有数据存储在可编辑的 JSON 文件中。
- **完全可定制**：直接在代码/数据中更改 logo、颜色、文本、模板和域名。
- **销售统计仪表板**：每日、每周、每月报告，带可视化图表。
